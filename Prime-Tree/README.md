# The Barbelo Prime Tree

A visual representation of prime numbers in which primes form a vertical trunk and composite numbers branch left and right — making prime gaps instantly visible as branch width.

## Primzahlen Baum II — A333489

Building on Primzahlen Baum I, this visualization overlays the integer sequence **OEIS A333489** onto the prime tree. A333489 contains all numbers k for which the k-th composition in standard order is an *anti-run* (no two adjacent equal parts). These numbers are highlighted in **magenta** regardless of whether they are prime or composite.

Sequence begins: `0, 1, 2, 4, 5, 6, 8, 9, 12, 13, 16, 17, 18, 20, 22, 24, 25, 32, ...`

Reference: https://oeis.org/A333489
